What is Rate Limiting? APIs impose rate limits to protect the system from excessive requests.

Example Rate Limit Headers:

        {
          "X-RateLimit-Limit": 1000,
          "X-RateLimit-Remaining": 950,
          "X-RateLimit-Reset": 1678909876
        }
