What is cURL?

cURL (Client URL) is a **command-line tool** used for sending HTTP requests to APIs. It is widely used by developers, testers, and API writers for quick and effective API interaction.

Basic Syntax

The most basic cURL command:

curl https://api.example.com

This sends a simple **GET request** to the API.

Adding Headers

Use the -H flag to include request headers:

curl -X GET "https://api.example.com" -H "Authorization: Bearer your_token"

Sending Query Parameters

To include query parameters:

curl -X GET "https://api.example.com/users?limit=10&sort=desc"

Sending Data in a Request Body

Use the -d flag for JSON data in POST or PUT requests:

curl -X POST "https://api.example.com/users" -d '{"name":"John"}' -H "Content-Type: application/json"

Authentication Methods

API Key Authentication

curl -X GET "https://api.example.com" -H "Authorization: Bearer your_api_key"

Basic Authentication

curl -u username:password "https://api.example.com"

Handling File Uploads

To upload a file using cURL:

curl -X POST -F "file=@myfile.jpg" "https://api.example.com/upload"

Downloading Files

Download a file with cURL:

curl -O https://example.com/file.pdf

Handling API Errors

Use the -i flag to view response headers and status codes:

curl -i "https://api.example.com"

Making Requests Silent (Without Output)

Use -s to hide unnecessary output:

curl -s "https://api.example.com"

Following Redirects

Use -L to follow redirects:

curl -L "https://api.example.com/redirect"
