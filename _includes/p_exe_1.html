Step 1: Install & Test an API in Postman

Before we create our own API, let’s first learn how to **interact with an existing API** using Postman.

1️⃣ Install Postman

Download and install **Postman** from: https://www.postman.com/downloads/

2️⃣ Send a GET request in Postman

Open Postman and **send a GET request** to:

    https://jsonplaceholder.typicode.com/users

You should see a list of users in JSON format.

3️⃣ Send a POST request in Postman

Now, let's try **creating a new user**. In Postman, send a **POST request** to:

    https://jsonplaceholder.typicode.com/users

With the following JSON body:

    {
        "name": "John Doe",
        "email": "john@example.com"
    }

If successful, you’ll get a **201 Created** response with the new user.

Step 2: Create Your Own API Using Python (Flask)

Now, let’s build our own API in Python.

1️⃣ Install Python & Flask

If you don’t have Python installed, download it from: https://www.python.org/downloads/

Next, install Flask by running this command in **VS Code terminal**:

    pip install flask

2️⃣ Create a new Python file

Open **VS Code**, create a new file called app.py, and add this code:

    from flask import Flask, request, jsonify

    app = Flask(__name__)

    users = [
        {"id": 1, "name": "Alice", "email": "alice@example.com"},
        {"id": 2, "name": "Bob", "email": "bob@example.com"}
    ]

    @app.route('/users', methods=['GET'])
    def get_users():
        return jsonify(users)

    @app.route('/users', methods=['POST'])
    def add_user():
        new_user = request.json
        users.append(new_user)
        return jsonify(new_user), 201

    if __name__ == '__main__':
        app.run(debug=True)

3️⃣ Run Your API

In the **VS Code terminal**, run:

    python app.py

Your API will start at http://127.0.0.1:5000

Step 3: Test Your API in Postman

Send a GET request

In **Postman**, make a **GET request** to:

    http://127.0.0.1:5000/users

You should see a list of users.

Send a POST request

Use **Postman** to send a **POST request** with this JSON body:

    {
        "id": 3,
        "name": "Charlie",
        "email": "charlie@example.com"
    }

If successful, you’ll see **201 Created** and the new user added.
